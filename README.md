# BookStore Backend

Spring Boot REST API for a full-stack BookStore E-Commerce Application.

## Features

* User Registration & Login
* JWT Authentication & Authorization
* Product Management
* Category Management
* Shopping Cart
* Wishlist
* Order Management
* Customer Profile
* Reviews & Ratings
* Admin Dashboard APIs

## Tech Stack

* Java 17
* Spring Boot
* Spring Security
* JWT
* MySQL
* JPA / Hibernate
* Maven

## Run Locally

1. Configure MySQL database
2. Update application.yml
3. Run:

mvn spring-boot:run
